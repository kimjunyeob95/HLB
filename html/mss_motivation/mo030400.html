보스는 모든 것을 알고 있고, 리더는 질문을 한다.

팀장은 부서 전체의 문서관리를 총괄하며 문서 권한 및 유통을 관리하는 책임자입니다.
팀원들이 회사의 자산인 문서를 보다 효율적으로 활용하며 스마트워크 할 수 있도록 적극적으로 지원해주시기 바랍니다.

AutoDocs 적용 부서
1. 팀의 대표 부서문서관리자를 지정해주시기 바랍니다.

- 대표 부서문서관리자는 팀장의 책임하에 AutoDocs 부서함 전체 권한을 조율하고 팀원들의 시스템 사용 전반을 관장하는 관리자입니다.

2. 부서장은 아래 항목에 관한 승인 권한이 있습니다.
  • ① 보안 등급 변경 : 파일의 보안 등급 변경 시 승인
  • ② 보안문서 해제 반출 : 파일 암호화 해제 반출 시 승인
  • ③ 취합요청 자료 제출 : 취합폴더에 파일 제출 시 승인
  • ④ 출장문서함 : 출장문서함 신청 시 승인
  • ⑤ 프로젝트 문서함 : 프로젝트 문서함 생성 시 승인
  • ⑥ 부서간 파일 복사 : 겸직 부서함 혹은 프로젝트 문서함과 부서함 간 파일 복사 시 승인
  • ⑦ 부서간 문서 이관 : 조직 변경으로 인한 부서간 문서 이관 시 승인
  • ⑧ 임시 부서 겸직 : 임시 겸직 신청 시 승인
AutoDocs 미적용 부서
1. 부서장은 팀원의 보안문서 반출 승인 권한이 있습니다.
  • - “보안문서해제승인시스템” 통해 팀 소유 문서 반출 승인
  • - 팀원 보안문서 반출 시 사유의 적절성을 검토하고 최종 승인/결재
  • - 결재 완료 시 팀원에게 자동으로 메일 발송(결재 결과)
